# R017

사용자스토리: 사용자가 알림을 탭하면 해당 메모 상세 페이지로 이동할 수 있다 (https://www.notion.so/3223b5d3bb7281fe8647eb95a31f640f?pvs=21)
업무영역: 프론트엔드
유형: 기능

## 요약

알림 탭 시 해당 메모 상세 페이지로 이동하는 딥링크 연동 기능.

## 수용 기준 (Acceptance Criteria)

- [ ]  알림 수신 후 탭 시 앱이 포그라운드로 전환됨
- [ ]  해당 메모 상세 페이지로 자동 네비게이션
- [ ]  메모가 삭제된 경우 '존재하지 않는 메모' 안내 화면 표시
- [ ]  앱이 종료된 상태에서 알림 탭 시에도 정상 동작

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 푸시 알림 payload에 `memoId` 포함하여 발송
2. 알림 탭 이벤트 리스너 등록 (포그라운드/백그라운드/종료 상태 각각 처리)
3. `memoId`를 파라미터로 메모 상세 페이지로 네비게이션
4. 메모 상세 API 404 응답 시 에러 화면 렌더링

## 제약 사항 (Constraints)

- 알림 payload에 반드시 `memoId` 필드 포함 필요
- iOS/Android 각 플랫폼 알림 탭 처리 방식 상이 — 플랫폼별 구현 필요
