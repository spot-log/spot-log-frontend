# R012

사용자스토리: 사용자가 개인 메모 반경에 진입할 때 자동 알림을 받을 수 있다 (https://www.notion.so/3223b5d3bb72816daa7cd231e735199e?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

개인 메모 반경에 사용자가 진입했을 때 자동으로 로컬 푸시 알림을 발송하는 지오펜싱 기능.

## 수용 기준 (Acceptance Criteria)

- [ ]  앱 실행 중 또는 백그라운드 상태에서 개인 메모 반경 진입 시 알림 발송
- [ ]  알림 내용에 메모 제목 포함
- [ ]  동일 메모에 대해 반경 이탈 후 재진입 시에만 재알림 (R017 참고)
- [ ]  알림 권한 미허용 시 알림 발송 시도 없음

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 앱 초기화 시 위치 권한 및 알림 권한 요청
2. 기기 위치 변경 이벤트 구독 (백그라운드 포함)
3. 활성 개인 메모 목록을 로컬 캐시로 유지
4. 현재 위치와 각 메모 중심 좌표 간 거리 계산
5. 거리 ≤ 메모 반경인 경우 로컬 알림 발송
6. 진입 상태 플래그 관리 — 이탈 후 재진입 시에만 재알림

### 백엔드

1. 사용자 개인 메모 목록 API에서 반경(radius) 필드 포함 반환
2. 서버 측 지오펜싱 트리거는 클라이언트에서 처리 — 별도 서버 푸시 불필요

## 제약 사항 (Constraints)

- 위치 추적 주기: 배터리 소모 최소화를 위해 OS 권장 설정 준수
- 백그라운드 위치 권한 별도 요청 필요 (iOS: Always, Android: Background Location)
