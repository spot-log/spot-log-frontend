# R007

사용자스토리: 사용자는 지도에서 공개 메모를 확인할 수 있다 (https://www.notion.so/3223b5d3bb7281ceafe0d9e27c42dcb6?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

공개 메모를 지도 뷰에서 조회하는 기능. 사용자가 지도 화면에서 타인의 공개 메모를 핀 형태로 확인할 수 있음.

## 수용 기준 (Acceptance Criteria)

- [ ]  지도 뷰 진입 시 현재 뷰포트 내 공개 메모 핀이 렌더링됨
- [ ]  핀 탭 시 메모 제목 및 미리보기(본문 앞 30자)가 표시됨
- [ ]  만료된 공개 메모는 지도에 노출되지 않음
- [ ]  공개 메모 핀과 개인 메모 핀은 시각적으로 구분됨
- [ ]  지도 이동/줌 시 뷰포트 내 메모 목록 갱신됨

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 지도 뷰 컴포넌트에서 뷰포트 변경(이동/줌) 이벤트 감지
2. 뷰포트 경계 좌표를 파라미터로 공개 메모 조회 API 호출 (디바운스 적용)
3. 응답 데이터를 지도 위에 마커로 렌더링
4. 마커 탭 이벤트 핸들러 구현 — 미리보기 말풍선 표시
5. 개인/공개 메모 마커 색상 또는 아이콘 분기 처리

### 백엔드

1. 뷰포트 또는 현재 위치 + 반경 기반 공개 메모 조회 API 구현
2. 만료일이 현재 시각 이전인 메모는 쿼리 결과에서 제외
3. 응답 필드: 메모 ID, 제목, 본문 앞 30자(summary), 위도/경도, 만료일

## 제약 사항 (Constraints)

- 만료된 메모는 어떠한 경우에도 공개 조회 결과에 포함되지 않음
- 지도 API 호출 최소화를 위해 뷰포트 변경 이벤트에 디바운스 적용 권장
