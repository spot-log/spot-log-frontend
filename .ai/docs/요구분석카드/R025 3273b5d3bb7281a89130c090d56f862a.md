# R025

사용자스토리: 사용자가 북마크한 타인의 공개 메모를 별도 탭에서 관리하며 만료 시 자동 제거된다 (https://www.notion.so/3223b5d3bb7281adb8e7e7375a04d72f?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

북마크한 타인의 공개 메모를 별도 탭에서 관리하는 기능. 만료된 메모는 북마크 목록에서 자동 제거됨.

## 수용 기준 (Acceptance Criteria)

- [ ]  내 메모 또는 별도 화면에 '북마크' 탭 존재
- [ ]  북마크한 공개 메모 목록이 조회됨
- [ ]  만료된 북마크 메모는 목록에서 자동 제거됨
- [ ]  북마크 해제 가능
- [ ]  빈 상태 시 '북마크한 메모가 없습니다' 안내

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 북마크 탭 진입 시 북마크 목록 API 호출
2. 결과를 카드 리스트로 렌더링
3. 각 아이템에 북마크 해제 버튼 제공
4. 빈 상태 UI 처리

### 백엔드

1. `GET /bookmarks` API 구현 — 인증 사용자 기준 북마크 목록 반환
2. 만료된 메모(`expiredAt < NOW()`)는 응답에서 제외 (또는 자동 삭제 스케줄러 적용)
3. 응답 필드: 메모 ID, 제목, 작성자, 만료일, 거리(위치 파라미터 제공 시)
4. 북마크 해제: `DELETE /memos/{id}/bookmark` (R019 준용)

## 제약 사항 (Constraints)

- 본인이 작성한 메모의 북마크 조회 포함 여부는 정책에 따름
- 만료 메모 자동 제거는 DB 스케줄러 또는 조회 시 필터링 중 선택
