# R004

사용자스토리: 사용자가 현재 위치 기반으로 텍스트 메모를 작성하고 지도에 핀으로 저장할 수 있다 (https://www.notion.so/3223b5d3bb72811eb0b9e29cf6e438f7?pvs=21)
스프린트 배정: 1차
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

현재 위치 기반으로 텍스트 메모를 작성하고 지도에 핀으로 저장하는 핵심 메모 생성 기능. 위치는 GPS 자동 설정 또는 검색으로 변경 가능함.

## 수용 기준 (Acceptance Criteria)

- [ ]  메모 생성 버튼 탭 시 작성 폼 진입
- [ ]  제목(선택), 본문(필수, 1~1000자), 유형(개인/공개), 위치 입력 가능
- [ ]  기본 위치는 현재 GPS 좌표로 자동 설정
- [ ]  장소 검색으로 위치 변경 가능
- [ ]  저장 완료 시 해당 좌표에 핀이 지도에 표시됨
- [ ]  본문 미입력 또는 1000자 초과 시 저장 불가

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 메모 생성 버튼 탭 → 작성 폼 화면 이동
2. 입력 필드 구현: 제목, 본문(글자 수 카운터 포함), 유형 선택, 위치 표시
3. GPS 현재 위치 자동 획득 및 지도 미리보기 표시
4. 장소 검색 컴포넌트 연동 (R010 참고)
5. 저장 버튼 탭 → 유효성 검사 → 메모 생성 API 호출
6. 성공 시 지도 화면으로 복귀 및 핀 표시

### 백엔드

1. 인증된 사용자로부터 제목, 본문, 위도/경도, 유형, 반경 수신
2. 본문 길이 검증: 1자 미만 또는 1000자 초과 시 400 반환
3. 제목 미입력 시 본문 앞 10자를 제목으로 자동 설정 (R024 참고)
4. 메모 엔티티 저장 후 생성된 메모 객체 반환

## 제약 사항 (Constraints)

- 인증된 사용자만 메모 생성 가능 — 미인증 시 401 반환
- 좌표 정밀도: 소수점 7자리 이상 유지
- 본문 최대 길이: 1000자 (멀티바이트 문자 기준 character 단위)
