# R027

사용자스토리: 사용자가 트리거 반경을 50m / 100m / 200m / 500m 중 선택하여 설정할 수 있다 (https://www.notion.so/50m-100m-200m-500m-3223b5d3bb7281ddadd3c8842ba7dc1b?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

개인 메모의 지오펜싱 트리거 반경을 사용자가 직접 설정하는 기능. 50m / 100m / 200m / 500m 중 선택.

## 수용 기준 (Acceptance Criteria)

- [ ]  메모 작성 폼 또는 상세 화면에 반경 선택 UI 존재
- [ ]  50m / 100m / 200m / 500m 4가지 옵션 제공
- [ ]  선택된 반경이 메모에 저장됨
- [ ]  저장된 반경이 지오펜싱 진입 판정에 사용됨 (R012 연계)

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 반경 선택 컴포넌트 구현 (라디오 버튼 또는 세그먼트)
2. 선택된 반경 값을 `radius` 필드로 API 요청에 포함
3. 기존 메모 수정 시 현재 반경 값으로 초기화
4. 지오펜싱 로직에서 메모별 `radius` 값 사용 (R012 연계)

### 백엔드

1. 메모 생성/수정 API에서 `radius` 필드 수신 (단위: m)
2. 허용값 검증: 50, 100, 200, 500 외 값 입력 시 400 반환
3. 메모 엔티티에 `radius` 필드 저장
4. 내 메모 조회 API 응답에 `radius` 포함

## 제약 사항 (Constraints)

- 허용 반경 값: 50, 100, 200, 500 (m) — 허용 외 값은 서버에서 거부
- 기본값: 100m (미선택 시 적용)
