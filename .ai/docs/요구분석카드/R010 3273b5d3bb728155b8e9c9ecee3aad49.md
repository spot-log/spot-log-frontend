# R010

사용자스토리: 사용자가 장소명 또는 주소 검색으로 메모 위치를 지정할 수 있다 (https://www.notion.so/3223b5d3bb72812db959fedc11a665ec?pvs=21)
스프린트 배정: 1차
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

장소명 또는 주소 텍스트 검색을 통해 메모 위치를 지정하는 기능. GPS 자동 설정 외 수동 위치 지정 수단 제공.

## 수용 기준 (Acceptance Criteria)

- [ ]  메모 작성 폼 내 검색 입력창에서 장소명/주소 검색 가능
- [ ]  검색 결과 목록에서 항목 선택 시 해당 좌표로 위치 설정
- [ ]  선택된 위치가 지도 및 입력 필드에 반영됨
- [ ]  검색 결과가 없는 경우 '검색 결과 없음' 메시지 표시

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 위치 검색 입력창 컴포넌트 구현 (디바운스 300ms 적용)
2. 지도 SDK의 Geocoding API 또는 Places API 호출
3. 검색 결과 목록 렌더링
4. 항목 선택 이벤트 → 위도/경도 추출 → 상위 폼 상태 업데이트
5. 지도 뷰 카메라를 선택된 좌표로 이동 및 핀 표시

### 백엔드

1. 위치 검색은 클라이언트에서 외부 지도 SDK 직접 호출 방식으로 처리 — 별도 서버 API 불필요
2. 메모 저장 시 최종 선택된 위도/경도 수신 및 저장 (기존 메모 생성 API 활용)

## 제약 사항 (Constraints)

- 지도 SDK 제공자의 Geocoding API 일일 호출 한도 준수
- 좌표 정밀도: 소수점 7자리 이상 유지
