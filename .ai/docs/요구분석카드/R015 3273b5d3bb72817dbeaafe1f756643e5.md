# R015

사용자스토리: 사용자가 공개 메모를 거리 가까운 순으로 정렬된 리스트로 확인할 수 있다 (https://www.notion.so/3223b5d3bb728163bb48ef528774d89d?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

주변 공개 메모를 현재 위치 기준 거리 가까운 순으로 정렬된 리스트로 확인하는 기능.

## 수용 기준 (Acceptance Criteria)

- [ ]  현재 위치 기준 일정 반경 내 공개 메모를 거리 오름차순으로 표시
- [ ]  각 아이템에 메모 제목, 거리, 만료일 표시
- [ ]  만료된 메모는 목록에서 제외
- [ ]  위치 권한 미허용 시 안내 메시지 표시

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 현재 위치 좌표 획득 (GPS)
2. 현재 위치 + 반경 파라미터로 공개 메모 조회 API 호출
3. 응답 데이터를 거리 오름차순으로 정렬하여 리스트 렌더링
4. 각 아이템에 거리(m/km), 제목, 만료일 표시

### 백엔드

1. `GET /memos/nearby` API에 `lat`, `lng`, `radius` 파라미터 수신
2. ST_Distance 또는 Haversine 공식으로 거리 계산
3. 만료일 필터링 적용 (`expiredAt >= NOW()`)
4. 거리 오름차순 정렬 후 반환 (각 메모에 distance 필드 포함)

## 제약 사항 (Constraints)

- 기본 조회 반경: 500m
- 최대 반환 건수: 50건 (페이지네이션 추후 적용)
