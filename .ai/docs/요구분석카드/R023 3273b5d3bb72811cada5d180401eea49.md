# R023

사용자스토리: 사용자가 공개 메모 알림을 ON/OFF로 설정할 수 있다 (기본값: OFF) (https://www.notion.so/ON-OFF-OFF-3223b5d3bb72815a9f03dad1a1b4a473?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

공개 메모 알림 ON/OFF 설정 기능. 기본값은 OFF이며 사용자가 명시적으로 활성화해야 함.

## 수용 기준 (Acceptance Criteria)

- [ ]  설정 화면에 '공개 메모 알림' 토글 존재
- [ ]  기본값: OFF
- [ ]  ON 전환 시 위치 및 알림 권한 요청 (미허용 시 토글 복원)
- [ ]  OFF 상태에서는 공개 메모 반경 진입 시 알림 미발송

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 설정 화면에 토글 컴포넌트 구현
2. 토글 ON 시 위치/알림 권한 상태 확인 → 미허용 시 권한 요청
3. 권한 거부 시 토글 OFF 복원 및 안내 메시지 표시
4. 설정값을 로컬 스토리지 또는 서버에 저장
5. 지오펜싱 감지 로직에서 해당 설정값 참조하여 공개 메모 알림 분기

### 백엔드

1. 사용자 설정 API에 `publicMemoNotification` 필드 추가 (`PATCH /users/me/settings`)
2. 기본값: `false`
3. 클라이언트에서 조회 시 해당 필드 반환

## 제약 사항 (Constraints)

- 개인 메모 알림(R012)은 별도 설정으로 관리됨 — 본 기능과 독립적
- 설정 기본값 OFF는 서버 및 클라이언트 양쪽에서 동일하게 보장
