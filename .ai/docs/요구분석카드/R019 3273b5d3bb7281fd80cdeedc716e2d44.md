# R019

사용자스토리: 사용자가 공개 메모 상세(제목, 전문, 작성일, 만료일)를 확인하고 북마크할 수 있다 (https://www.notion.so/3223b5d3bb72819c9e01eb6dc3ec42bf?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

공개 메모 상세 정보(제목, 전문, 작성일, 만료일) 조회 및 북마크 기능.

## 수용 기준 (Acceptance Criteria)

- [ ]  공개 메모 탭 시 상세 화면 진입
- [ ]  제목, 전문, 작성일, 만료일 표시
- [ ]  북마크 버튼 존재 및 북마크 상태 시각적 구분(토글)
- [ ]  북마크 추가/해제 즉시 반영
- [ ]  본인 메모에도 북마크 가능 여부는 정책 결정에 따름

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 메모 상세 API 호출 및 화면 렌더링
2. 북마크 버튼 상태를 API 응답의 `isBookmarked` 필드로 초기화
3. 북마크 버튼 탭 → 북마크 토글 API 호출
4. 성공 시 버튼 상태 즉시 갱신

### 백엔드

1. `GET /memos/{id}` API에 `isBookmarked` 필드 포함 반환 (인증 사용자 기준)
2. `POST /memos/{id}/bookmark` — 북마크 추가
3. `DELETE /memos/{id}/bookmark` — 북마크 해제
4. 중복 북마크 요청 시 409 또는 멱등 처리
5. 만료된 메모 조회 시에도 상세 내용 반환 (만료 상태 명시)

## 제약 사항 (Constraints)

- 비인증 사용자는 북마크 기능 사용 불가 (401 반환)
- 삭제된 메모 북마크 요청 시 404 반환
