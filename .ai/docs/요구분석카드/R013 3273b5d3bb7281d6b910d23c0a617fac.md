# R013

사용자스토리: 사용자가 메모 작성 시 개인 또는 공개 유형을 선택할 수 있다 (https://www.notion.so/3223b5d3bb728155853ddd3d60160a8e?pvs=21)
스프린트 배정: 1차
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

메모 작성 시 개인(비공개) 또는 공개 유형을 선택하는 기능. 유형에 따라 노출 범위 및 추가 설정 항목이 달라짐.

## 수용 기준 (Acceptance Criteria)

- [ ]  메모 작성 폼에 '개인' / '공개' 유형 선택 UI 존재
- [ ]  '공개' 선택 시 유효 기간 설정 항목이 폼에 노출됨
- [ ]  '개인' 선택 시 유효 기간 설정 항목이 폼에서 숨겨짐
- [ ]  선택된 유형이 API 요청에 정확히 반영됨
- [ ]  유형 미선택 시 저장 불가(필수값 처리)

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 세그먼트 컨트롤 또는 라디오 버튼으로 유형 선택 컴포넌트 구현
2. 유형 상태 변경 감지 → 유효 기간 섹션 조건부 렌더링
3. 폼 제출 시 선택된 유형을 `type` 필드로 API 요청 포함

### 백엔드

1. 메모 생성 API에서 `type` 필드 필수값 검증 (`PRIVATE` / `PUBLIC`)
2. `type` 미포함 또는 허용 외 값 입력 시 400 반환
3. 유형에 따른 추가 필드 검증 수행 (공개 시 `expiredAt` 필수 — R008 준용)

## 제약 사항 (Constraints)

- 메모 저장 후 유형 변경 기능은 현재 스코프 외
- `type` 허용값: `PRIVATE`, `PUBLIC` (대소문자 구분)
