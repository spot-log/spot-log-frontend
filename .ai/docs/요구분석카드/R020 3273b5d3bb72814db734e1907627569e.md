# R020

사용자스토리: 사용자가 내 메모를 최근 작성순 / 만료 임박순 / 거리 가까운 순으로 정렬할 수 있다 (https://www.notion.so/3223b5d3bb72815593b1e7b4e7417293?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

내 메모 목록을 최근 작성순 / 만료 임박순 / 거리 가까운 순으로 정렬하는 기능.

## 수용 기준 (Acceptance Criteria)

- [ ]  내 메모 목록 화면에 정렬 옵션 선택 UI 존재
- [ ]  최근 작성순 / 만료 임박순 / 거리 가까운 순 3가지 옵션 제공
- [ ]  정렬 변경 시 목록 즉시 재정렬
- [ ]  거리순 정렬 선택 시 현재 위치 권한 필요 — 미허용 시 안내 메시지 표시
- [ ]  '만료됨' 탭에서는 거리순 정렬 비활성화 또는 미제공

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 정렬 옵션 선택 컴포넌트 구현 (드롭다운 또는 세그먼트)
2. 정렬 변경 시 선택된 `sort` 파라미터로 API 재호출
3. 거리순 선택 시 현재 위치 좌표를 파라미터에 추가

### 백엔드

1. 내 메모 목록 API에 `sort` 파라미터 지원
    - `CREATED_AT_DESC`: 최근 작성순
    - `EXPIRE_SOON`: 만료 임박순 (`expiredAt` 오름차순, NULL 후순)
    - `DISTANCE`: 거리 가까운 순 (위도/경도 파라미터 필요)
2. 거리순 요청 시 `lat`, `lng` 필수값 검증 — 미포함 시 400 반환
3. 정렬 기본값: `CREATED_AT_DESC`

## 제약 사항 (Constraints)

- 개인 메모는 만료일 없음 — 만료 임박순 정렬 시 개인 메모는 후순 배치
