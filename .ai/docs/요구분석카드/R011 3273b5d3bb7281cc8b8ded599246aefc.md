# R011

사용자스토리: 사용자가 내 메모를 개인 / 공개 / 만료됨 탭으로 분류하여 관리할 수 있다 (https://www.notion.so/3223b5d3bb7281879af3c763b399ef15?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

내 메모 목록을 개인 / 공개 / 만료됨 탭으로 분류하여 관리하는 기능. 메모 유형 및 만료 여부에 따라 탭 구분 제공.

## 수용 기준 (Acceptance Criteria)

- [ ]  내 메모 화면에 '개인' / '공개' / '만료됨' 탭 표시
- [ ]  각 탭 선택 시 해당 유형의 메모만 렌더링
- [ ]  만료된 공개 메모는 '공개' 탭이 아닌 '만료됨' 탭에만 노출
- [ ]  각 탭에 메모 건수 뱃지 표시

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 탭 컴포넌트 구현 (개인 / 공개 / 만료됨)
2. 탭 변경 시 해당 파라미터로 내 메모 조회 API 재호출
3. 응답 데이터를 메모 카드 리스트로 렌더링
4. 각 탭 헤더에 건수 표시

### 백엔드

1. 내 메모 목록 API에 `type` 파라미터 지원 (`PRIVATE` / `PUBLIC` / `EXPIRED`)
2. `EXPIRED`: `type=PUBLIC` AND `expiredAt < NOW()` 조건
3. `PUBLIC`: `type=PUBLIC` AND `expiredAt >= NOW()` 조건
4. `PRIVATE`: `type=PRIVATE` 조건
5. 각 조건에 맞는 메모만 필터링하여 반환

## 제약 사항 (Constraints)

- 본인 소유 메모에 한해 조회 가능
- 탭별 페이지네이션 적용 (기본 20건)
