# R026

사용자스토리: 사용자가 현재 위치 500m 이내 공개 메모를 지도 뷰에서 핀으로 확인하고 미리볼 수 있다 (https://www.notion.so/500m-3223b5d3bb7281f7892cf0362319c0e3?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

현재 위치 500m 이내 공개 메모를 지도 뷰에서 핀으로 확인하고 미리볼 수 있는 기능. R007과 연계되나 반경 기준이 명시적으로 500m로 고정됨.

## 수용 기준 (Acceptance Criteria)

- [ ]  지도 뷰에서 현재 위치 기준 500m 이내 공개 메모 핀 노출
- [ ]  핀 탭 시 제목 및 본문 미리보기 표시
- [ ]  만료된 메모는 핀에서 제외
- [ ]  현재 위치가 변경되면 핀 목록 갱신

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 위치 변경 이벤트 구독 (디바운스 적용)
2. 현재 위치 + 반경 500m 파라미터로 주변 메모 API 호출
3. 응답 데이터를 지도 마커로 렌더링
4. 마커 탭 시 미리보기 말풍선 표시 (제목 + 본문 앞 30자)
5. 상세 조회는 미리보기에서 '자세히 보기' 버튼으로 R019 진입

### 백엔드

1. `GET /memos/nearby?lat={}&lng={}&radius=500` API 활용 (R015 준용)
2. 반경 기본값을 500m로 명세
3. 응답에 미리보기용 `summary`(본문 앞 30자) 필드 포함

## 제약 사항 (Constraints)

- 조회 반경 고정값: 500m (변경 시 R015와 함께 수정)
- 위치 정확도에 따라 표시 범위 오차 발생 가능
