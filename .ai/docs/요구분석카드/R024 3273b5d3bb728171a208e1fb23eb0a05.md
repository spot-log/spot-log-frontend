# R024

사용자스토리: 사용자가 메모 제목 미입력 시 본문 앞 10자가 제목으로 자동 설정된다 (https://www.notion.so/10-3223b5d3bb7281c2b4c2db1a3a72a546?pvs=21)
업무영역: 백엔드
유형: 기능

## 요약

메모 제목 미입력 시 본문 앞 10자를 제목으로 자동 설정하는 기능. 서버 측에서 처리함.

## 수용 기준 (Acceptance Criteria)

- [ ]  제목 없이 메모 저장 요청 시 본문 앞 10자가 제목으로 저장됨
- [ ]  본문이 10자 미만인 경우 전체 본문이 제목으로 설정됨
- [ ]  제목 입력 시 자동 설정 로직 미동작
- [ ]  저장된 메모 조회 시 자동 설정된 제목이 반환됨

## 작업 단계 (Implementation Steps)

### 백엔드

1. 메모 생성 요청에서 `title` 필드 null 또는 빈 문자열 여부 확인
2. 조건 충족 시 `content` 앞 10자를 `title`로 할당
3. `content` 길이가 10 미만인 경우 `content` 전체를 `title`로 사용
4. 최종 `title` 값으로 메모 엔티티 저장

## 제약 사항 (Constraints)

- 자동 제목 설정은 서버에서만 처리 (클라이언트 미리보기 불필요)
- 멀티바이트 문자(한글 등) 기준 10자 — byte가 아닌 character 단위
