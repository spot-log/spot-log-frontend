# R016

사용자스토리: 사용자가 만료된 공개 메모를 '만료된 메모' 탭에서 확인할 수 있다 (https://www.notion.so/3223b5d3bb7281dca512e13269f0b8d3?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

내 메모 관리 화면의 '만료된 메모' 탭에서 만료된 공개 메모 목록을 확인하는 기능.

## 수용 기준 (Acceptance Criteria)

- [ ]  '만료됨' 탭에서 현재 시각 기준 만료된 본인의 공개 메모 목록 조회
- [ ]  각 아이템에 제목, 만료일, 작성일 표시
- [ ]  만료됨 탭이 비어있는 경우 '만료된 메모가 없습니다' 안내 문구 표시
- [ ]  R009(재공개/삭제) 액션 진입 가능

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. '만료됨' 탭 활성화 시 만료 메모 목록 API 호출
2. 결과를 카드 리스트로 렌더링
3. 빈 상태(empty state) UI 처리
4. 각 카드에 재공개/삭제 액션 버튼 노출 (R009 연계)

### 백엔드

1. 내 메모 목록 API에 `type=EXPIRED` 파라미터 처리 (R011 준용)
2. 조건: 본인 소유 AND `type=PUBLIC` AND `expiredAt < NOW()`
3. 만료일 오름차순 정렬하여 반환

## 제약 사항 (Constraints)

- 개인 메모는 만료됨 탭 대상 아님
- 본인 소유 만료 메모만 조회 가능
