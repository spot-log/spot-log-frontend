# R021

사용자스토리: 사용자가 GPS 기반 현재 위치를 메모 위치로 자동 설정할 수 있다 (https://www.notion.so/GPS-3223b5d3bb72816ab49ecfd0432ac16d?pvs=21)
스프린트 배정: 1차
업무영역: 프론트엔드
유형: 기능

## 요약

GPS를 통해 현재 위치를 메모 위치로 자동 설정하는 기능. 메모 작성 폼 진입 시 기본값으로 적용.

## 수용 기준 (Acceptance Criteria)

- [ ]  메모 작성 폼 진입 시 현재 위치 좌표가 기본 위치로 자동 설정됨
- [ ]  지도에 현재 위치 핀이 표시됨
- [ ]  위치 권한 미허용 시 안내 메시지 표시 및 수동 위치 설정으로 대체
- [ ]  위치 획득 중 로딩 인디케이터 표시

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 메모 작성 폼 마운트 시 Geolocation API 호출
2. 위치 획득 성공 시 `lat`, `lng` 상태 초기화 및 지도 카메라 이동
3. 위치 획득 실패 또는 권한 거부 시 에러 핸들링 및 수동 위치 입력 안내
4. 로딩 상태 UI 처리

## 제약 사항 (Constraints)

- 위치 권한은 '앱 사용 중' 최소 권한으로 요청
- GPS 정확도 허용 오차: 반경 50m 이내 권장
