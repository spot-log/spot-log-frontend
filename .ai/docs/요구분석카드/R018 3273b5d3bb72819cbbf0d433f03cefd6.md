# R018

사용자스토리: 사용자가 계정 탈퇴 시 본인이 작성한 모든 메모(공개 포함)가 삭제된다 (https://www.notion.so/3223b5d3bb7281b5b364ecd6954b19c9?pvs=21)
업무영역: 백엔드, 프론트엔드
유형: 기능

## 요약

계정 탈퇴 기능. 탈퇴 시 본인이 작성한 모든 메모(공개 포함)가 삭제됨.

## 수용 기준 (Acceptance Criteria)

- [ ]  설정 화면에서 계정 탈퇴 메뉴 접근 가능
- [ ]  탈퇴 전 경고 다이얼로그 표시 ('모든 메모가 삭제됩니다' 안내 포함)
- [ ]  탈퇴 확인 시 계정 및 모든 메모 즉시 삭제
- [ ]  탈퇴 완료 후 로그인 화면으로 이동
- [ ]  타인이 북마크한 본인의 공개 메모도 삭제됨

## 작업 단계 (Implementation Steps)

### 프론트엔드

1. 설정 화면에 계정 탈퇴 항목 구현
2. 탈퇴 버튼 탭 → 경고 다이얼로그 표시
3. 확인 → 탈퇴 API 호출 → 성공 시 로컬 인증 토큰 제거 및 로그인 화면 이동

### 백엔드

1. `DELETE /users/me` API 구현
2. 인증 토큰 검증 후 사용자 ID 추출
3. 해당 사용자의 모든 메모 삭제 (공개/개인 구분 없이)
4. 삭제된 메모에 연결된 북마크 레코드 연쇄 삭제
5. 사용자 계정 삭제
6. 트랜잭션으로 처리 — 중간 실패 시 롤백

## 제약 사항 (Constraints)

- 탈퇴 처리는 단일 트랜잭션 내에서 완료되어야 함
- 탈퇴 후 동일 소셜 계정으로 재가입 가능 (신규 계정으로 처리)
