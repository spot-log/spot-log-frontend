\## 목차



1\. 메모 작성

2\. 장소 설정

3\. 메모 유형 (개인 / 공개)

4\. 위치 기반 알림

5\. 공개 메모 탐색

6\. 메모 유효성 관리

7\. 알림 설정

8\. 계정 및 메모 관리



\---



\## 1. 메모 작성



\### 1-1. 기본 작성



| 항목 | 내용 |

| --- | --- |

| 기능명 | 메모 작성 |

| 설명 | 텍스트 기반 메모를 작성하고 특정 좌표에 저장한다 |

| 입력값 | 제목(선택), 본문(필수, 1 \~ 1000글자), 장소, 유형(개인/공개) |

| 출력값 | 저장된 메모 객체 |

|  |  |



\*\*상세 요건\*\*



\- 본문은 최소 1자 이상 입력 시 저장 가능

\- 제목 미입력 시 본문 앞 10자를 제목으로 자동 설정

\- 메모 작성 시 현재 위치를 기본값으로 자동 설정하며, 사용자가 직접 변경 가능

\- 작성 완료 후 해당 좌표에 핀으로 표시됨



\### 1-2. 메모 삭제



| 항목 | 내용 |

| --- | --- |

| 기능명 | 메모 삭제 |

| 설명 | 기존에 저장된 메모를 삭제한다 |

| 접근 조건 | 본인이 작성한 메모에 한함 |



\*\*상세 요건\*\*



\- 삭제 시 확인 다이얼로그 표시 후 즉시 삭제



\---



\## 2. 장소 설정



\### 2-1. 좌표 지정 방식



| 방식 | 설명 |

| --- | --- |

| 현재 위치 자동 설정 | GPS 기반 현재 좌표를 메모 위치로 자동 지정 |

| 지도에서 직접 선택 | 지도를 탭하여 원하는 좌표를 수동 지정 |

| 검색으로 지정 | 장소명 또는 주소 검색 후 선택 |



\### 2-2. 트리거 반경 설정



| 항목 | 내용 |

| --- | --- |

| 기본값 | 100m |

| 설정 범위 | 50m / 100m / 200m / 500m 중 선택 |

| 설명 | 사용자가 설정 반경 내에 진입 시 메모가 표시됨 |



\---



\## 3. 메모 유형



\### 3-1. 개인 메모



| 항목 | 내용 |

| --- | --- |

| 공개 범위 | 본인에게만 표시 |

| 유효 기간 | 없음 (사용자가 직접 삭제 전까지 유지) |

| 알림 | 반경 진입 시 자동 알림 |

| 삭제 조건 | 사용자 직접 삭제 |



\*\*사용 예시\*\*



\- 마트 방문 시 구매 목록 확인

\- 헬스장 도착 시 오늘의 운동 루틴 표시

\- 도서관 입장 시 찾아볼 책 목록 표시



\### 3-2. 공개 메모



| 항목 | 내용 |

| --- | --- |

| 공개 범위 | 해당 좌표 반경에 진입한 모든 사용자 |

| 유효 기간 | 작성자 직접 설정 또는 기본값 자동 적용 |

| 알림 | 사용자 알림 설정에 따라 수신 여부 결정 |

| 탐색 | 앱 내 '주변 메모' 탭에서 수동 확인 가능 |



\*\*사용 예시\*\*



\- 가게 입구 위치 안내

\- 주차 팁 및 혼잡 시간대 공유

\- 숨겨진 명소 소개



\---



\## 4. 위치 기반 알림



\### 4-1. 트리거 조건



| 조건 | 설명 |

| --- | --- |

| 개인 메모 | 메모가 저장된 좌표 반경 진입 시 즉시 알림 |

| 공개 메모 | 알림 수신 설정이 ON인 경우, 반경 진입 시 알림 발송 |



\### 4-2. 알림 형태



| 항목 | 내용 |

| --- | --- |

| 표시 방식 | 가능하면 앱 실행, 불가능하면 푸시 알림 + 앱 내 팝업 오버레이 |

| 표시 내용 | 메모 제목, 없으면 본문 앞 20자 |

| 액션 | 알림 탭 시 해당 메모 상세 페이지로 이동 |



\### 4-3. 중복 알림 방지



| 항목 | 내용 |

| --- | --- |

| 조건 | 동일 메모에 대해 반경 이탈 후 재진입 시에만 재알림 |

| 목적 | 같은 장소에 머무는 동안 반복 알림 방지 |



\---



\## 5. 공개 메모 탐색



\### 5-1. 주변 메모 보기



| 항목 | 내용 |

| --- | --- |

| 기능명 | 주변 공개 메모 탐색 |

| 설명 | 현재 위치 기준 일정 반경 내 공개 메모를 지도 또는 리스트로 확인 |

| 기본 탐색 범위 | 500m 이내 |



\*\*상세 요건\*\*



\- 지도 뷰: 핀으로 공개 메모 위치 표시, 탭 시 메모 미리보기

\- 리스트 뷰: 거리 가까운 순으로 정렬

\- 메모는 남은 시간이 짧을수록 투명도를 낮춰 시각적으로 표시



\### 5-2. 메모 상세 보기



| 항목 | 내용 |

| --- | --- |

| 표시 내용 | 메모 제목, 메모 전문, 작성 일시, 만료 일시 |

| 액션 | 저장(북마크) |

\## 6. 메모 유효성 관리



\### 6-1. 개인 메모



| 항목 | 내용 |

| --- | --- |

| 유효 기간 | 없음 |

| 만료 조건 | 사용자 직접 삭제 시에만 삭제 |



\### 6-2. 공개 메모



| 항목 | 내용 |

| --- | --- |

| 유효 기간 설정 | 작성 시 직접 설정 가능 (1일 / 1주 / 1개월 / 3개월 / 직접 입력) |

| 기본값 | 미설정 시 30일 후 자동 만료 |

| 만료 전 시각적 처리 | 생성일 → 만료일 비율만큼 투명화 |

| 만료 처리 | 만료 시 공개 노출에서 제거, 작성자 메모함에는 '만료됨' 상태로 보관 |



\### 6-3. 만료 메모 관리



| 항목 | 내용 |

| --- | --- |

| 접근 경로 | 내 메모 > 만료된 메모 |

| 가능한 액션 | 삭제, 재공개 (내부적으로는 생성) |



\---



\## 7. 알림 설정



| 설정 항목 | 설명 | 기본값 |

| --- | --- | --- |

| 개인 메모 알림 | 개인 메모 반경 진입 시 알림 ON/OFF | ON |

| 공개 메모 알림 | 주변 공개 메모 반경 진입 시 알림 ON/OFF | OFF |

\## 8. 계정 및 메모 관리



\### 8-1. 계정



| 항목 | 내용 |

| --- | --- |

| 가입 방식 | 소셜 로그인 (Google) |

| 탈퇴 | 계정 삭제 시 본인 메모 전체 삭제 (공개 메모 포함) |



\### 8-2. 내 메모 관리



| 항목 | 내용 |

| --- | --- |

| 내 메모 목록 | 개인 / 공개 / 만료됨 탭으로 분류 |

| 정렬 기준 | 최근 작성순 / 만료 임박순 / 거리 가까운 순 |

| 북마크한 메모 | 타인의 공개 메모 중 저장한 항목 별도 관리, 만료 시 자동 제거 |
